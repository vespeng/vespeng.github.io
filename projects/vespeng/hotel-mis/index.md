# 酒店管理信息系统

基于 SSM 框架开发的酒店管理信息系统，适用练习 demo 或课程设计，满足基础的包括对住客、房间和会员信息的管理功能，同时支持导出房间及会员信息至 excel。前端实现采用传统 Layui-Qadmin 模板，界面设计简洁美观，代码结构清晰易懂，欢迎下载学习或二次开发。

## 安装步骤

1. 克隆或下载本项目源码，将其导入到 IDE 中
2. 配置 Tomcat 服务器，将应用上下文路径（Context Path）设置为：`/`
3. 修改 resources/properties/jdbc.properties 文件，配置数据库连接信息
4. 导入 hotel.sql 数据库文件到您的 MySQL 数据库中

> ps：JDK 版本 1.8、MySQL 版本 5.7 及以上、Tomcat 版本 8.0 及以上
