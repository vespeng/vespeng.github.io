## 配置环境

1. 安装好之后须添加如下环境变量：

   | 变更名        | 变量值       | 说明                                      |
      | ---------- | --------- | --------------------------------------- |
   | **GOPATH** | E:\gowork | Go语言的工作目录，存放自己编写的 .go 文件、项目、包、编译的二进制文件等 |
   | **GOROOT** | D:\go | Go的安装路径                                 |

   在 Path 路径下，新增 _%GOROOT%\bin_
   > ps: GOPATH 的值，需要根据实际情况按需配置，默认路径为C盘用户目录下

2. 开启go mod 及配置国内代理：

   ```shell
   go env -w GO111MODULE=on
   go env -w GOPROXY=https://goproxy.cn,direct
   ```
